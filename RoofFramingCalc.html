Hip / Valley — Between Two Different Pitches

Valley line = where both roof planes reach the same height
Height A = y × PitchA ÷ 12  |  Height B = x × PitchB ÷ 12  → set equal
Direction vector = ( PitchA, PitchB )  |  parametric t along valley line
Hits Ridge A at t = RunA ÷ PitchB  |  Hits Ridge B at t = RunB ÷ PitchA
Terminates at whichever ridge it reaches first (smaller t)
Plan angles: atan( PitchB ÷ PitchA ) from Wall A  |  atan( PitchA ÷ PitchB ) from Wall B
Backing angles differ each side: atan( sin(planAngle) × tan(pitchAngle) )
