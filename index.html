User Guide

Construction Calculator
A full construction math calculator with feet-inches support, just like the Construction Master Pro.
Feet-Inches Entry

Type a number, then press Feet to lock it as feet. Type the next number, press Inch for inches.

All math (+, -, ×, ÷) works in decimal feet internally, but the display shows feet-inches too.

12 Feet 6 Inch + 3 Feet 2 Inch =
= 15' 8"
Pitch / Rise / Run

Enter any two of the three values, and the calculator solves for the third.

Type a number, then press Pitch, Rise, or Run. When two are set, the third appears automatically.

Example: enter 8 Pitch then 12 Run → Rise is solved.

Memory

Stor — saves current value to memory

Rcl — recalls the stored value

M+ — adds current value to memory (running total)

An M indicator appears in the display when memory has a value.

Swipe Navigation

Swipe left or right anywhere on the screen to switch between calculators. The dots at the bottom show which calculator you're on.

Roof Framing
Calculate rafter lengths, angles, and cuts for common rafters, hip/valley at equal or unequal pitches.
Common Rafter

Run — horizontal distance from eave to ridge centerline (in feet)

Pitch — rise per 12 inches of run (e.g., 8 means 8/12 pitch)

Outputs: Rise, Rafter Length, Slope Angle, Plumb Cut, Seat Cut

Run = 12.5 ft, Pitch = 8/12
Rafter Length = 15.02 ft
Hip / Valley

Same Pitch — both sides of a 90° corner have the same pitch. Enter Run and Pitch.

Unequal Pitch — two different pitches meeting at a corner. Enter Run and Pitch for each side (A and B).

Roof Area on Slope
Convert flat (plan view) area to actual sloped area for material ordering. Handles multiple roof sections at different pitches.
How It Works

A sloped roof has more surface area than its footprint. The slope factor = √(1 + (pitch/12)²).

At 8/12 pitch, the slope factor is 1.2019 — meaning 20% more area than flat.

Results include roofing squares (1 square = 100 sq ft) with a 10% waste estimate.

Adding Multiple Areas

Click + Add Area for each roof section. Enter length, width, and pitch for each. Different sections can have different pitches.

40' × 25' @ 8/12 = 1201.9 sq ft
30' × 20' @ 6/12 = 670.8 sq ft
Total = 18.73 squares (21 with waste)
Stair Layout
Calculate risers, treads, stringer length, and check IRC code compliance. Supports landings.
Inputs

Total Rise — floor-to-floor height in inches

Target Riser — desired riser height (default 7.5")

Tread Depth — horizontal depth of each tread (default 10")

Headroom — available clearance in inches

Landings

Set Number of Landings to split the stair into flights. Each landing adds its depth to the total run.

The results show a flight-by-flight breakdown with risers, treads, and stringer length per flight.

108" total rise, 7.5" riser, 1 landing
2 flights: 7R/7T + 7R/6T, 13.83 ft span
Code Compliance

Results are color-coded per IRC R311.7:

Within code
Out of range

Riser: 4"–7¾" | Tread: ≥10" | Headroom: ≥80"

Concrete & Material
Calculate concrete volume in cubic yards and bags for slabs, footings, and columns.
Slab / Pad

Enter Length (ft) × Width (ft) × Thickness (inches).

20' × 30' × 4" thick
= 8.15 cu yd (with 10% waste)
Multiple Footings

The Footings tab lets you add multiple footing entries. Choose a preset size (16"×8", 20"×10", etc.) or select Custom to enter your own width and depth.

Click + Add Footing for each footing run. Enter the length in feet for each. The total is calculated at the bottom.

Column / Sono Tube

Enter Diameter (inches) and Height (inches). Uses πr²h formula.

Waste Factor

The 10% waste checkbox adds 10% to the calculated volume. This accounts for spillage, over-excavation, and uneven forms. Uncheck it for exact volume only.

Feet-Inches Converter
Live bidirectional conversion between decimal feet, feet-inches-fractions, and metric.
How It Works

Edit any field and all others update instantly — no calculate button needed.

The green dot indicates live mode is active.

Three Formats

Decimal Feet — e.g., 12.5417

Feet-Inches-Fraction — separate fields for feet, whole inches, and fraction (to 1/16")

Metric — meters, centimeters, and millimeters (all linked)

12.5417 ft = 12' 6 1/2" = 3.8227 m
